What I Create

감각을 반응형 경험으로 바꾸는 오디오비주얼 시스템

Objects can become interfaces.

제품, 오브젝트, 소재, 공간을 터치와 센서 기반 인터페이스로 전환해
실시간 사운드와 비주얼이 반응하는 경험을 만듭니다.

01

인터랙티브 설치

Interactive Installations

터치, 센서, 오브젝트 입력을 실시간 사운드와 비주얼 반응으로 연결하는 설치 시스템입니다.

02

브랜드 & 이벤트 비주얼 시스템

Brand & Event Visual Systems

브랜드 팝업, 청음회, 런칭 이벤트, 대형 스크린, 문화행사에 적용 가능한 오디오 반응형 비주얼과 공간형 미디어 콘텐츠입니다.

03

워크숍 / 크리에이티브 테크놀로지

Workshops / Creative Technology

TouchDesigner, 터치 인터랙션, 센서 기반 프로토타이핑, 참여형 오디오비주얼 시스템을 다루는 아티스트 주도형 워크숍입니다.

Selected Work

작업 가능성을 보여주는 프로젝트

01 / Audio-Reactive Visual System

김준수 (XIA) — 정규앨범 'Gravity' 청음회

공식 앨범 선공개 청음회를 위해 제작한 30분 길이의 오디오 반응형 비주얼 시스템.

앨범 프리뷰, 아티스트 쇼케이스, 런칭 이벤트, 음악 중심 브랜드 모먼트로 확장 가능합니다.

02 / Large-scale Media Art Visuals

PayPal—Melbourne Fashion Festival

대형 패션 / 문화행사 맥락에서 상영된 미디어아트 비주얼과 스크린 기반 이벤트 콘텐츠.

대형 스크린, 문화행사, 페스티벌, 공간형 미디어 연출에 적용할 수 있습니다.

03 / Interactive Installation

The Meta Rose Project

장미를 촉각 인터페이스로 사용해 실시간 사운드와 비주얼을 작동시키는 인터랙티브 오디오비주얼 설치 작업.

브랜드 팝업, 인터랙티브 부스, 감각적 사진 모먼트, 체험형 소매 공간에 적용할 수 있습니다.

04 / Creative Technology Workshop

TouchCollective / Melbourne Fringe Workshop

TouchDesigner, 전도성 터치, 꽃과 식물, 집단적 사운드, 실시간 비주얼을 다룬 참여형 워크숍.

TouchDesigner 워크숍, 팀빌딩 워크숍, 문화예술 교육 프로그램으로 구성할 수 있습니다.
